自动生成论文多级标题并排版正文

# 自动生成论文多级标题并排版正文

  • 以大学生的睡眠文化作为论文内容，展示如何设置论文中的各种格式，文章内容来自网上搜集。

# 1 - 论文多级标题及正文

# 1.1 - 多级标题

一篇论文需要有总体框架 (opens new window)，也就是先做什么，后做什么，这些步骤通过多级标题来体现，而本科生毕业设计论文中多级标题的级别一般为三，也就是最多是三级标题，如果在三级标题下还有需要序号的内容，一般用小序号完成。多级标题的紧跟内容称为标题后续样式，下方的正文等也需要进行对应的设置

# 1.2 - 正文

正文指在多级标题下编写的文本，它是论文的主体内容，一篇论文中约90%的内容都属于正文，其余为标题、图表、题注和下标注等

# 2 - 如何自动生成

对于标题， 千万不要自己手动输入 ，比如自己输入 第一章 绪论 ，然后调整字体格式……，千万不要，因为一旦某个章节需要增删，那么将会导致链式反应的修改，因此我们需要使用Word的自定义多级标题去完成

# 2.1 - 格式约定

以下是约定的多级标题和正文格式（当你的要求并非如此时，可以跟随步骤进行修改）：

  • 多级标题的级数为三级
  • 一级标题：用作章节， 要求黑体、小二、居中对齐 ，例如第一章、第二章……
  • 二级标题：用作章节下的排序， 要求黑体、小三、左顶格对齐、包含一级标题 ，例如1.1、1.2……
  • 三级标题：用作二级标题下的排序， 要求宋体、四号、左顶格对齐、包含一和二级标题 ，例如1.1.1、1.1.2……
  • 标题的后续格式为正文，要求宋体、小四、两端对齐、首行缩进2字符、1.5倍行距
  • 多级标题和正文中如有西文，统一字体为Times New Roman

# 2.2 - 设置多级标题

接下来进行多级标题自动生成的实操，也就是完成上述格式中粗体字的要求，可以仔细观察每一个步骤，所有步骤的进行都与格式约定有关

# 2.2.1 - 设置标题样式

  1. 【开始】-【段落】-【多级列表】-【定义新的多级列表】 ，需要关注的为下图红色块

  2. 设置一级标题样式： 【选择左侧的1】-【编号格式中输入“第”和“章”】-【编号样式选择中文大写序号】-【链接样式中选择标题1】-【库级别选择级别1】-【编号之后选择空格】 ，按照下图顺序操作


3. 设置一级标题字体： 续上一个窗口，【字体】-【中文字体选黑体】-【西文字体选Times New Roman】-【字号选小二】-【确定】

4. 设置二级标题样式：续上一个窗口，【选择左侧的2】-【链接样式选择标题2】-【级别选择级别2】-【勾选正规形式编号】-【对齐位置填0】-【编号后选择空格】

5. 设置二级标题字体：续上一个窗口，【字体】-【中文选黑体】-【西文选Times New Roman】-【字号选小三】-【确定】

6. 设置三级标题样式： 续上一个窗口，【选择左侧的3】-【链接样式选择标题3】-【级别选择级别3】-【勾选正规形式编号】-【对齐位置填0】-【编号后选择空格】

7. 设置三级标题字体： 续上一个窗口，【字体】-【中文选宋体】-【西文选Times New Roman】-【字号选四号】-【确定】

8. 最后点击【确定】，此时一个多级标题的框架已经出现

# 2.2.2 - 设置标题后续样式

我们还有后续样式需要设置，下面进行：【开始】-【样式】-【标题1】-【右键修改】

  1. 修改一级标题后续样式：【后续段落选择正文缩进】-【格式字体选黑体】-【字号选小二】-【取消加粗】-【选择居中对齐】-【格式选段落】-【行距选择1.5倍】-【确定】

  2. 修改二级标题后续样式：【右击标题2选择修改】-【后续段落选择正文缩进】-【格式字体选黑体】-【字号选小三】-【取消加粗】-【选择左对齐】-【格式选段落】-【行距选择1.5倍】-【确定】

  3. 修改三级标题后续样式：【右击标题3选择修改】-【后续段落选择正文缩进】-【格式字体选宋体】-【字号选四号】-【取消加粗】-【选择左对齐】-【格式选段落】-【行距选择1.5倍】-【确定】

  4. 至此，多级标题已经设置完成，但是论文中还需要一个和一级标题的级别一致但不是序号显示的标题，这个标题用于论文的摘要、目录、致谢等内容，我们可以这样设置：【右击标题选择修改】-【后续段落选择正文缩进】-【格式字体选宋体】-【字号选四号】-【取消加粗】-【选择居中对齐】-【格式选段落】-【行距选择1.5倍】-【确定】



最后，我们修改的后续样式有以下的红块区域：

# 2.3 - 设置正文格式

  1. 修改正文格式：在样式中 【正文右键修改】-【字体选宋体】-【字号选四号】-【对齐选两端对齐】-【切换为西文字体选择Times New Roman】

  2. 修改正文行距：续上一个窗口，【行距选择1.5倍】-【确定】

  3. 最后点【确定】，正文样式设置完成

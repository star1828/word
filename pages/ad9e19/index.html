# 小修改和小问题

# 1 - 小修改

无论你觉得自己的毕业论文一稿有多么完美，在导师眼中都千仓百孔，导师一定会提出很多要求，其中就包含很多格式要求，小白在此只列举若干个，其它的格式要求可以参考其余教程的步骤去更改：

# 1.1 - 修改正文字体和字号

【开始】-【样式】-【正文】-【右键：修改】-【选择字体和字号】

# 1.2 - 修改正文行距

【开始】-【样式】-【正文】-【右键：修改】-【格式】-【段落】-【行距】-【多倍行距：输入需要修改的行距比如1.25】

# 1.3 - 修改标题字体和字号

【开始】-【样式】-【某级标题】-【右键：修改】-【选择字体和字号】

# 2 - 小问题

# 2.1 - 当发现首行没有缩进2字符

千万不要敲空格 ，用 【段落下三角】-【特殊：首行】-【缩进值：2字符】 完成

# 2.2 - 当发现有些删不到的页

千万不要死按backspace ，尝试按几次 delete

# 2.3 - 当发现整句英文被挤到下一行

千万不要敲空格 ，点击所在段落，然后 【段落下三角】-【中文办板式】-【勾选：允许西文xxxxx】

# 2.4 - 当发现删不掉的小黑点

千万不要按backspace ，小黑点是标记符号，实际是不会打印出来的，如果不想看见，可以在 【文件】-【选项】-【显示】-【取消一些选项】
